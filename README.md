# Swift-UIKit

## Overview
`Swift-UIKit` is a Swift project designed for testing UIKit.

## Requirements
- iOS 15.6+
- Xcode 14+
- Swift 5+

## Features
- [x] Text (UILabel)
- [x] Button
- [x] PopUp Button
- [x] PullDown Button
- [x] Switch Button
- [x] Toggle Button
- [x] TextField
- [x] TableView
- [ ] CollectionView
- [ ] StackView
- [ ] Progress
- [ ] Slider
- [ ] Stepper
- [ ] PageControl
- [x] Segment
- [x] Picker
- [x] Alert
